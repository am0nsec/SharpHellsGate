Parameters
ProcessHandleA handle for the process for which the mapping should be done.
BaseAddressA pointer to a variable that will receive the base address of the allocated region of pages.
ZeroBitsThe number of high-order address bits that must be zero in the base address of the section view.
RegionSizeA pointer to a variable that will receive the actual size, in bytes, of the allocated region of pages.
AllocationTypeA bitmask containing flags that specify the type of allocation to be performed for the specified region of pages.
ProtectA bitmask containing page protection flags that specify the protection desired for the committed region of pages.
Returns
NtAllocateVirtualMemory returns either STATUS_SUCCESS or an error status code.
